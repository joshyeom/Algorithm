### 🔗 문제: [LeetCode] 1470. Shuffle the Array

#### 1. 문제 핵심 요약
- 배열을 순회하면서 i, i + n 요소를 재배치
- 예시 [1,2,3,4,5,6,7,8] / n = 4 => [1,5,2,6,3,7,4,8]

#### 2. 해결 로직 및 전략
- 배열을 순회하며 새로운 빈 배열에 push

#### 3. 코드 복잡도 분석
- 시간 복잡도: O(n)

#### 4. 사용된 주요 문법 및 자료구조
- 베열

#### 5. 회고 및 개선할 점
- 공간복잡도를 익힐 수 있었음

```js
ar shuffle = function(nums, n) {
    const MAX_VAL = 1024; // 1000보다 큰 2의 거듭제곱 (10비트 확보)

    // 1단계: Encoding (하나의 인덱스에 두 값을 저장)
    // nums[i]에 x와 y를 동시에 저장합니다.
    for (let i = 0; i < n; i++) {
        // 현재 x값에 y값을 10비트 왼쪽으로 밀어서 합칩니다.
        // x_i = nums[i], y_i = nums[i + n]
        nums[i] = nums[i] | (nums[i + n] << 10);
    }

    // 2단계: Decoding (뒤에서부터 원래 위치에 배치)
    // 뒤에서부터 채워야 정보를 덮어쓰지 않습니다.
    for (let i = n - 1; i >= 0; i--) {
        const x = nums[i] & 1023;       // 하위 10비트 추출 (x)
        const y = nums[i] >> 10;        // 상위 10비트 추출 (y)

        nums[2 * i + 1] = y;            // 홀수 인덱스에 y 배치
        nums[2 * i] = x;                // 짝수 인덱스에 x 배치
    }

    return nums;
};
```
